UC00604 · Programar para a web, na vertente frontend (cliente-side)

Bem-vindo(a) à unidade de
programação web frontend

Nesta unidade vais aprender a criar páginas e interfaces web no lado do cliente, usando HTML, CSS, JavaScript e boas práticas de acessibilidade e RGPD.

4,5 créditos Nível 5 Componente Tecnológica

O que vais aprender

  • Definir a estrutura de páginas web com HTML.
  • Aplicar folhas de estilo CSS para criar layouts responsivos.
  • Integrar scripts em JavaScript no frontend.
  • Conceber e implementar o frontend de uma aplicação web.
  • Aplicar regras de acessibilidade e proteção de dados.

Sobre a unidade

A UC00604 integra a formação tecnológica do curso de Técnico/a Especialista em Tecnologias e Programação de Sistemas de Informação e centra-se na construção de interfaces web modernas, responsivas e acessíveis.

Competências em destaque

  • Planeamento de páginas e estruturação de conteúdos.
  • Programação de páginas web com HTML, CSS e JavaScript.
  • Utilização de bibliotecas frontend.
  • Aplicação de normas e regulamentos (incluindo RGPD).

Onde se aplica

  • Empresas de desenvolvimento e serviços de informática.
  • Lojas e espaços comerciais de informática.
  • Serviços de apoio e assistência técnica.
  • Organismos da Administração Pública com serviços digitais.
