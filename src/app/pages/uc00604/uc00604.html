UC00604

Programar para a web, na vertente frontend (cliente-side)

Pontos de crédito: 4,5 · Componente Tecnológica · Nível 5

Finalidade da Unidade de Competência

Desenvolver a camada de apresentação de aplicações web, criando páginas e interfaces no lado do cliente, com HTML, CSS e JavaScript, garantindo usabilidade, acessibilidade e cumprimento das normas legais (incluindo proteção de dados).

Realizações

  • Definir a estrutura de páginas web utilizando linguagem de marcação HTML.
  • Construir páginas web organizadas, aplicando folhas de estilo CSS.
  • Integrar scripts na criação de páginas web para suportar funcionalidades.
  • Conceber e implementar o frontend de aplicações web.

Critérios de Desempenho

  • Respeitar orientações e especificações técnicas na construção das páginas.
  • Aplicar regras e boas práticas de programação na implementação do frontend.
  • Garantir que a interface é responsiva e funcional em diferentes dispositivos.
  • Identificar e corrigir erros, testando e validando as páginas web.
  • Cumprir regulamentos e normas, incluindo o RGPD e legislação aplicável.

Conhecimentos

  • Evolução do desenvolvimento web e principais tecnologias associadas.
  • Conceitos e organização de páginas web, criação, publicação e responsividade.
  • Estrutura de páginas web, etiquetas mais frequentes, imagens e hiperligações.
  • Utilização de tabelas para organização de informação em páginas web.
  • Conceção de formulários para recolha e submissão de dados.
  • Fundamentos de CSS e utilização de folhas de estilo.
  • Conceitos base de JavaScript aplicado ao desenvolvimento web.
  • Finalidade e uso de bibliotecas frontend.
  • Princípios de acessibilidade em páginas web.
  • Função de servidores virtuais no desenvolvimento e testes.
  • Regulamento Geral sobre a Proteção de Dados (RGPD).
  • Normas e regulamentos relevantes para o desenvolvimento de aplicações web.

Aptidões

  • Interpretar manuais, guiões e tutoriais técnicos.
  • Elaborar o mapa do site sob a forma de diagrama.
  • Construir wireframes para prototipar layouts de páginas.
  • Aplicar regras de organização de páginas na definição da estrutura em HTML.
  • Integrar diferentes elementos numa página usando marcadores HTML.
  • Criar hiperligações internas e externas em páginas web.
  • Escrever e organizar folhas de estilo respeitando a sintaxe de CSS.
  • Associar e integrar CSS em documentos HTML.
  • Compreender a lógica e estrutura da linguagem JavaScript.
  • Desenvolver scripts em JavaScript para suportar o frontend.
  • Aplicar regras de acessibilidade nas interfaces web.
  • Criar e configurar um servidor virtual para testes.
  • Realizar testes, identificar problemas e corrigi-los.

Atitudes

  • Assumir responsabilidade pelos resultados do trabalho.
  • Demonstrar autonomia na execução das tarefas.
  • Manter persistência perante dificuldades técnicas.
  • Atuar com ética profissional.
  • Revelar iniciativa e proatividade.
  • Exercer sentido crítico sobre as soluções implementadas.
  • Organizar o trabalho de forma planeada.
  • Respeitar regras, normas e procedimentos da organização.

Contexto de Utilização da Competência

  • Empresas de desenvolvimento e serviços de informática.
  • Lojas e estabelecimentos dedicados a produtos de informática.
  • Serviços de apoio e assistência técnica.
  • Organismos da Administração Pública com serviços digitais.

Recursos

  • Manuais, guiões e tutoriais técnicos.
  • Equipamentos com acesso à Internet.
  • Editor de texto ou bloco de notas.
  • Ambiente de desenvolvimento configurado.
  • Servidor virtual para testes e alojamento.
